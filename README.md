## 🚀 在线体验

访问：[https://sdti-demo.netlify.app](https://sdti-demo.netlify.app)

## 📱 技术栈

- 纯前端 HTML/CSS/JavaScript
- 响应式设计（桌面端 + 移动端）
- 无需后端，无需数据库

## 🛠️ 本地运行

直接在浏览器中打开 `sdti.html` 即可。

## 📄 测试原理

基于 **15维人格评估模型**（5个模型 × 3个维度）：
- 每个维度2道题，共30题
- 每题3个选项（1-3分）
- 分数归类为 L(低)/M(中)/H(高)
- 通过向量距离计算匹配最接近的类型
